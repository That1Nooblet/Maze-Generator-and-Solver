# Maze-Generator-and-Solver
Visualize multiple different methods of maze solving using DFS, BFS, and A*.

CONTROLS:
Q - Quit/Exit
G - Maze Generation 1 (randomly create walls)
H - Maze Generation 2 (DFS-assisted maze generation)
R - Reset/Clear
Space - Toggle Between Build and Solve Mode
Left Click (During Build Mode) - Place or Delete a wall
Left Click (During Solve Mode) - Set starting point
Right Click - Set ending point
P - Skip to path retracing animation

Numbers to switch between graph traversal methods (USE BEFORE LEFT CLICKING):
1 - DFS
2 - BFS
3 - A*
4 - Weighted A*
5 - Bidirectional BFS
6 - Bidirectional A*
